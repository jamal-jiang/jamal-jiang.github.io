Python-一等函数

发表于 2018-02-19 | 分类于 Python |

在Python中，函数是一等对象。

一等对象的定义如下：

  • 在运行时创建
  • 能赋值给变量或数据结构中的元素
  • 能作为参数传递给函数
  • 能作为函数的返回结果
